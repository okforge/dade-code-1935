# Dade County Building Code (1935) — okforge demo wiki

**Browse it live: <https://okforge.github.io/dade-code-1935/>**

This is a demo knowledge base produced by [okforge](https://github.com/okforge):
the 1935 Miami-Dade building code (a public record), scanned pages in, a
citation-backed wiki out — per-document summaries, cross-document concept and
entity pages, and `(p. N)` page citations throughout. Built as a static site
with [Quartz](https://quartz.jzhao.xyz/).

This repo holds only the generated site. The pipeline that produced it:
[okforge](https://github.com/okforge/okforge) (engine) ·
[okforge-vision-ocr](https://github.com/okforge/okforge-vision-ocr) (VLM OCR) ·
[okforge-webui](https://github.com/okforge/okforge-webui) (web UI + job queue).
